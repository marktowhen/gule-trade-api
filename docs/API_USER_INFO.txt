
1.1)通过uid查询个人信息
	
	方法：GETA
	资源路径：/api/userinfo/{uid}(与评论表相关联的id)
	请求参数类型：json
	请求参数：uid
	响应参数类型：json
	响应结果：{
		code:"",
		message:"",
		body:

		{
			
			UID:"与个人资料表关联的id"
