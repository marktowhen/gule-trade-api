评论
1.1）保存评论的信息
	方法：PUT
	资源路径：/api/comments/list
	请求参数类型：json
	请求参数:{
		GID:"商品的id"
		ImgID:"商品对应图片的id"
		replyComment:"回复的内容"
		goodsComment:"商品的评价"
		commentGrade:"评价的级别"（int）共10分一个星2分
		goodsService:"评价服务"
		serviceGrade："服务级别"（int）共10分一个星2分
		commentStatus:"评论的状态"（int ）1：未评论  2：已评论
		
	}
	响应结果的类型：json
	响应结果：{
		code:"",
		message:"",
		body:{
			GID:"商品的id"
			ImgID:"商品对应图片的id"
			replyComment:"回复的内容"
			goodsComment:"商品的评价"
			commentGrade:"评价的级别"（int）共10分一个星2分
			goodsService:"评价服务"
			serviceGrade："服务级别"（int）共10分一个星2分
			commentStatus:"评论的状态"（int ）1：未评论  2：已评论
		}
	}
